# Tick3t — Product Vision

**Version:** 1.0  
**North star:** Africa's Event Operating System

---

## Mission

Build the most complete Event Operating System in Africa. Tick3t is not just a ticket-selling website — it is the infrastructure that powers the entire event industry. Every organizer, venue owner, promoter, artist, vendor, sponsor, ambassador, and attendee should be able to operate from one platform, while **RedFace Pay** powers all commerce behind the scenes.

## Core philosophy

Traditional ticketing companies stop after selling a ticket.

Tick3t continues until the event is finished, everyone has been paid, reports are generated, customers are retained, and the next event is launched.

Every feature must answer:

> **Does this make running an event easier?**

## Product boundary

| Product | Owns |
|---------|------|
| **Tick3t** | Events, discovery, ticketing, check-in, organizers, staff, venues, suppliers, ambassadors, sponsors, event operations |
| **RedFace Pay** | Payments, settlements, wallets, payouts, invoices, merchant KYC, Paystack routing, financial infrastructure |

No payment logic is duplicated inside Tick3t. Tick3t consumes RedFace Pay through well-defined APIs and checkout deep-links.

## Five ecosystems (long-term)

1. **Attendee** — discover, buy, wallet, QR, refunds, reviews  
2. **Event organizer** — workspace, tickets, seating, sales, door ops, staff, finance  
3. **Venue marketplace** — profiles, availability, bookings, deposits  
4. **Service marketplace** — DJs, security, decor, catering, and more  
5. **Ambassador / affiliate** — referral links, commissions via RedFace Pay  

## Phase order (see ROADMAP.md)

1. Core operating system (production-ready event management)  
2. Event discovery  
3. Organizer growth tools  
4. Venue marketplace  
5. Event services marketplace  
6. Sponsor & vendor hub  
7. AI & intelligence  

## Together

**RedFace Pay** moves the money.  
**Tick3t** runs the event.  
Together they create a unified ecosystem that is significantly harder to replicate than a traditional ticketing service.
